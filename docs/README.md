# MAPORIA
**your telescope to Sri Lanka**

Welcome to the MAPORIA project documentation — the single source of truth for product requirements, architecture, and team workflow.

> MAPORIA helps users explore Sri Lanka physically — unlocking map regions automatically through GPS movement, saving geotagged memories, and sharing achievements.

## How to preview locally
1. Install docsify CLI (if not installed):
```bash
npm install -g docsify-cli
```

2. Navigate to the docs folder and start the server:
```bash
cd docs
docsify serve
```

3. Open your browser to `http://localhost:3000`
